    collecting behavioral data online.

    a course by todd m. gureckis

    {% endblock %} {% block pagecontent %}

    This class teaches the art, science, and technology necessary to collect useful behavioral data online for psychology, marketing, and computer science. It is composed of 11 video lectures, readings, and a variety of technical tutorials. Anyone can use these materials if they find them useful, but please reference this site.

    Spring 2014 syllabus

    Lectures

    1. Online data collection: A revolution for behavioral science? Or a curse?
    2. View from the other side - worker issues
    3. Using built-in Amazon tools, Qualtrics
    4. Open source software, git, and how to teach yourself stuff
    5. Introduction to psiTurk (download install, discuss features)
    6. Introduction to the psiTurk command line, run example experiment
    7. Debugging and developing using psiTurk
    8. Deploying using psiTurk, using advanced database solutions
    9. Getting your data out of database for analysis
    10. Designing for quality data
    11. Advanced topics - running experiments over multiple days/sessions, group experiments
